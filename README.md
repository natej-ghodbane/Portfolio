# 🚀 Natej Ghodbane — AI & Data Science Portfolio

Welcome to my personal portfolio website!  
Built with **Next.js**, **TypeScript**, and a fully responsive custom UI.

🌐 **Live Website:**  
👉 https://www.natejghodbane.me/en  

---

## 📁 Project Structure

```
portfolio/
│
├── app/
│   ├── components/        # Reusable UI components (Navbar, Hero, About, etc.)
│   ├── globals.css        # Global styles
│   ├── layout.tsx         # Root layout for the app
│   ├── page.tsx           # Homepage + sections
│
├── public/                # Static assets (images, icons, etc.)
│
├── .gitignore
├── eslint.config.mjs
├── next.config.ts
├── next-env.d.ts
├── package.json
├── postcss.config.mjs
├── tailwind.config.js 
├── tsconfig.json
└── README.md  
```

## 🚀 Deployment

The portfolio is deployed on **Vercel** and updates automatically on every push.

## 📦 Installation & Running Locally

```bash
git clone https://github.com/natej-ghodbane/Portfolio.git
cd portfolio
npm install
npm run dev
```

Then open:  
👉 http://localhost:3000
